# Green Launcher

This is an application manager/launcher intended for Windows 9x-based systems. I started this project because I wasn't able to find something native for Win9x that suited my needs. It's heavily based on Steam, Mame32 and D-Fend Reloaded GUIs, and it's coded in AutoIt3 scripting language, targeting version 3.2.12. It also makes use of SQLite 3.7.12, latest version to work with Windows 98 (at least)
