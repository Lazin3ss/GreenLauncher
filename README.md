# Green Launcher

This is an application manager/launcher intended for Windows 9x-based systems, though it should also work on Windows 10/11. I started this project because I wasn't able to find something native for Win9x that suited my needs. It's heavily based on Steam, MAME32 and D-Fend Reloaded, and it's coded in AutoIt3 scripting language, targeting version 3.2.12.1. It also makes use of SQLite 3.7.11, latest version to work with Windows 98 (at least).
